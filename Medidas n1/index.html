Caixa com PX - Medida Absoluta(Fixa)

Px é uma unidade absoluta, que tem o comportamento fixo para qualquer tamanho de tela. Isso significa que o tamanho do elemento não muda de acordo com o tamanho da tela. Ou seja, 100px será sempre 100px, seja em um computador ou celular.

Caixa % - Medida Relativa

Porcentagem é uma medida relativa(flexível) O tamanho do elemento muda conforme o tamanho do elemento pai. Por exemplo, se o pai tiver 100% e o filho usar 50%, ele ocupará metade.

Caixa com VW / VH - Medida Relativa

Viewport é uma palavra em inglês que significa algo como "área visível da tela". O tamanho do elemento muda conforme o tamanho da janela do navegador.
